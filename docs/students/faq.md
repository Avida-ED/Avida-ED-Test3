This page answers the most common student questions about using Avida-ED in a
course.

## Will Avida-ED run on my computer?

Avida-ED 4 is intended to be used in a current web browser on a desktop or
laptop computer. Avida-ED 3 historically ran in browsers on Macs, PCs, and
Chromebooks, and could sometimes run on tablets or phones, but the interface
was not designed for touch-first use.

For the current site, the practical guidance is:

- prefer a current desktop browser
- test on the device type required by your course
- avoid assuming that older browser-specific guidance from the legacy site is
  still the best recommendation

## Can I change environmental parameters during a run?

That depends on the version and the lesson workflow. In older Avida-ED versions,
many environmental controls became inactive once a run began, with only limited
controls such as pause remaining adjustable.

For current course use, follow the lesson instructions rather than assuming
every control should be changed mid-run.

## Why do older screenshots or instructions look different from what I see?

Some legacy materials were written for earlier standalone versions of Avida-ED.
If an older screenshot or instruction set does not match what you see, rely on
your instructor's current directions and the student pages in this site first.

## What should I do if my assignment result looks wrong?

Before assuming the software is wrong, check:

- whether you changed more than one variable
- whether you ran the comparison for the same duration
- whether you recorded the starting condition accurately
- whether your instructor expected a prepared setup instead of defaults

If the issue still looks like a software or browser problem, use
[Troubleshooting](../support/troubleshooting.md).
