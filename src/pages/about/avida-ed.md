---
title: What Is Avida-ED?
---

Avida-ED is an educational platform for learning evolution through experiments
with self-replicating digital organisms.

## Purpose

The software was created to make evolutionary processes easier to observe,
question, and discuss in classroom settings. Instead of treating evolution as a
sequence of fixed examples, Avida-ED gives learners a system they can manipulate
and investigate.

The legacy site repeatedly described Avida-ED as an award-winning educational
application developed at Michigan State University for undergraduate biology
courses. That remains the clearest short description of the project's public
purpose.

## How it is used

Instructors use Avida-ED to support:

- guided labs
- model lessons on specific evolutionary concepts
- open-ended research-style projects
- discussion of evidence, explanation, and scientific reasoning

Students use Avida-ED to:

- make predictions
- run experiments
- collect evidence
- interpret patterns of change in evolving populations

## What this site emphasizes now

This redesign treats Avida-ED primarily as a teaching and learning resource,
not as a software release portal. That means the main public pathways focus on:

- launching the current app
- getting students started on course work
- helping instructors choose and use curriculum
- providing support and transcript-backed guidance

Historical downloads and older version details still matter, but they now live
in supporting pages instead of driving the whole site structure.

## Educational emphasis

Avida-ED is most useful when the software supports a larger learning goal. The
site redesign underway in this repository reflects that priority by shifting
from a software-development presentation toward learner and instructor support.
