This page summarizes how Avida-ED versions should be interpreted on the site.

## Current guidance

- **Avida-ED 4** is the current default version.
- **Avida-ED 3** is legacy and should be used only when older course materials
  still depend on it.

## How to interpret version references

Use this rule of thumb across the site:

- if a page is written for current teaching use, assume Avida-ED 4
- if a lesson depends on older screenshots or workflows, label it clearly as
  legacy-oriented
- if an asset only exists for older releases, keep it available but route users
  back to current launch and support guidance first

## Why this matters

The legacy site accumulated release notes, download paths, and compatibility
messages over many years. Those records are still useful, but they can confuse
new users if they appear to compete with the main classroom path.

This page exists to interpret that history rather than reproduce every old
change log inline.

## What belongs here

- major version milestones
- compatibility notes that affect teaching materials
- context for older screenshots, downloads, and archived handouts

The full legacy change logs used symbol-coded entries for bug fixes, changes,
feature additions, and feature removals. That level of detail is still useful
for archive interpretation, but it should stay secondary to the simpler
version-choice guidance above.

Detailed historical release notes can remain available, but they should not
compete with the learner and instructor pathways that support current use.
