This page gathers the most important support paths for students and instructors
using Avida-ED.

## Common starting points

- If the app does not load, begin with the [Launch Avida-ED](/app) page and
  confirm you are using the current Avida-ED 4 link.
- If you need classroom materials, go to the
  [instructor curriculum hub](/curriculum).
- If you are a learner trying to begin an assignment, go to
  [For Students](/getstarted/students).
- If you need transcript-first help with a walkthrough or overview video, go to
  [Videos and Transcripts](/docs/videos).

## Instructor support

Instructors typically need help with:

- choosing a lesson sequence
- matching older handouts to newer versions of the application
- preparing devices before class
- finding model lessons and lab-manual materials

The instructor pathway on this site is being expanded to make those decisions
faster and to reduce dependence on older software-centered pages.

## Student support

Students usually need:

- a clear launch link
- orientation to the app interface
- guidance on where to begin in the assigned curriculum
- help understanding what to save, submit, or discuss after an activity

The student materials page is intended to collect those first-step resources in
one place.

## Classroom support workflow

When something goes wrong in a course setting, the fastest sequence is usually:

1. Confirm everyone is using the same version of the app.
2. Retry the task with the same browser and lesson steps.
3. Check the [Troubleshooting](/docs/support/troubleshooting) guide.
4. Use [Reporting Problems](/docs/support/reporting-problems) to capture the
   exact failure clearly.

## Download and archive support

If you need an older version, an offline copy, or historical reference
material, use:

- [Download / Offline](/download)
- [Version History](/about/version-history)
- [Vintage Archive](/about/vintage-archive)

Archive material remains available for compatibility and historical reference,
but current students and instructors should start from the learner and
curriculum paths above.
